These tests are executed through the Selenium IDE 2.4.0 FireFox plugin,
http://docs.seleniumhq.org/projects/ide/



Prior to running any/all test suites:
(you only have to do this once)

- Log in to the Admin app, in Firefox.

- Add 2 Administrators
	- rcv-admin
	- Alegeus
- Add 1 Role
	- name: rcv-admin, all-permissions, Entity:Choose rcv-admin from HTML SELECT 

	

- From the SeleniumIDE, load the test suite. It should be _suite.htm.
